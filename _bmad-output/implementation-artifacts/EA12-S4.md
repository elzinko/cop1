# Story EA12.4: Status discipline — remove BmadStatusReader (A6)

Status: done

## Story

As a **cop1 runtime**,
I want all file-level coupling to BMAD state (`sprint-status.yaml`, BMAD artifacts) removed from the codebase,
so that cop1 queries status exclusively via `/bmad-bmm-sprint-status` workflow invocation and maintains its own orchestrator state in `.cop1/orchestrator-state.yaml`.

## Acceptance Criteria

1. **AC1 — BmadStatusReader removed**
   - `BmadStatusReader` class (and its port/interface) deleted from the codebase.
   - All consumers migrated to use `/bmad-bmm-sprint-status` invocation via `invoke_bmad_command`.

2. **AC2 — Zero file-level coupling invariant**
   - `grep -r "sprint-status.yaml" packages/**/src/**/*.ts` returns zero results.
   - No runtime code reads, writes, or watches any BMAD artifact file directly.

3. **AC3 — Status query via BMAD workflow**
   - Supervisor queries status by invoking `/bmad-bmm-sprint-status` and interpreting the output.
   - Invocation is event-driven (after workflows that change status), not polling.

4. **AC4 — cop1 orchestrator-state.yaml**
   - cop1 writes its own operational state (current story, phase, blockers) into `.cop1/orchestrator-state.yaml` via internal API.
   - This file is cop1-owned, not BMAD-owned.

5. **AC5 — Tests**
   - Integration test verifying the grep invariant (AC2) — can be a script or test.
   - Unit test for the status query path via invoke_bmad_command.

## Dev Notes

### A6 pivot rationale

Per SCP 2026-04-15 §A6: D3's partial stance (read allowed, no write) was rejected. cop1 must have ZERO file-level coupling to BMAD state. Status queries route through `/bmad-bmm-sprint-status` workflow. This is slower (seconds, not milliseconds) but eliminates duplication with BMAD and allows the supervisor to adapt autonomously.

### Cost accepted

Status queries via workflow invocation are ~seconds vs ~ms for file reads. This cost is accepted because status queries are infrequent (event-driven, not polled) and the simplification eliminates an entire coupling surface.
