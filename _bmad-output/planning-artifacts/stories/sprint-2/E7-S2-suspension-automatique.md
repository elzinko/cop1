# Story E7-S2: Suspension automatique

Status: ready-for-dev

## Story
As a system administrator
I want the application to automatically suspend when RAM usage reaches 75%
so that system stability is maintained and resource exhaustion is prevented

## Acceptance Criteria
1. RAM monitoring system tracks memory usage continuously
2. Automatic suspension triggers when RAM reaches or exceeds 75%
3. Automatic resume triggers when RAM drops below 70%
4. Suspension state is logged and observable in system metrics
