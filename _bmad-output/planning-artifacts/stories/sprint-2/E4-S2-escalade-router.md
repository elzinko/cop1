# Story E4-S2: Escalade Router

Status: ready-for-dev

## Story
As a sprint core system
I want to implement an escalation router that routes blocages to appropriate agents by type
so that different blocage categories are handled by specialized agents

## Acceptance Criteria
1. Router maps blocage types to appropriate agent handlers
2. Routing logic is configurable and extensible
3. Multiple agents can handle the same blocage type
4. Router tracks routing decisions and agent assignments
