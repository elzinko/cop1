# Story E2-S11: Backlog Dashboard UI

Status: ready-for-dev

## Story
As a product manager
I want a web UI to manage backlog views
so that I can easily see Ideation/Grooming and Active Sprint stories

## Acceptance Criteria
- Web UI displays Ideation/Grooming view with story cards
- Web UI displays Active Sprint view with current stories
- Stories sortable and filterable by status, priority, size
- Real-time updates as sprint state changes
- Responsive design for various screen sizes

## Dev Notes
- Package: web
- Story Points: 5
