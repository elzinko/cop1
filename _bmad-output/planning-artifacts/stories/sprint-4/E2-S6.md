# Story E2-S6: INVEST Validator

Status: ready-for-dev

## Story
As a sprint planning system
I want to evaluate stories against INVEST criteria
so that story quality meets industry standards

## Acceptance Criteria
- INVESTValidator.check(snapshot) evaluates all 6 INVEST criteria
- Checks Independent (story can be developed independently)
- Checks Negotiable (flexible requirements)
- Checks Valuable (delivers user value)
- Checks Estimable (can be estimated by team)
- Checks Small (can be completed in sprint)
- Checks Testable (acceptance criteria clear)
- Returns detailed feedback per criterion
