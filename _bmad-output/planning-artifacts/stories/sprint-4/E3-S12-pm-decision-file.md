# Story E3-S12: PM Decision File

Status: ready-for-dev

## Story
As a PM agent system
I want PMAgent to persist decision questions
so that decision context is preserved for review

## Acceptance Criteria
- PMAgent.persistQuestion() saves decision questions to file
- Stores context and available options
- Records decision timestamp and rationale
- Supports querying historical decisions
- Enables audit trail for decision making
